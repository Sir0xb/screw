The "text" binding

字符串 or 返回字符串的函数(运行它) or ko对象

javascript

var theValue = ko.observable("theValue");
html

<!-- ko text: theValue --><!-- /ko -->
<div data-bind="text: theValue"></div>
Output

The "html" binding

javascript

var theValue = ko.observable("theValue");
html

<!-- ko html: theValue --><!-- /ko -->

Output


The "css" and "style" binding

属性值不能是变量

javascript

var theValue = ko.observable("theValue");
html

<style type="text/css">
.testRed { color: red; }
</style>
<div data-bind="css: { testRed: theValue }, text: theValue"></div>
<div data-bind="style: { color: theValue() ? 'blue' : 'green' }, text: theValue"></div>
Output


The "attr" binding

javascript

var theValue = ko.observable("theValue");
html

<a target="_blank" href="javascript:void(0);" data-bind="html: theValue">TheValue link</a>
