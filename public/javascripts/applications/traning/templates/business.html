The "value" binding

更新方式可以设置：valueUpdate 属性
  • “keyup” - updates your view model when the user releases a key  
  • “keypress” - updates your view model when the user has typed a key. Unlike keyup, this updates repeatedly while the user holds a key down  
  • “afterkeydown” - updates your view model as soon as the user begins typing a character. This works by catching the browser’s keydown event and handling the event asynchronously. This does not work in some mobile browsers.
javascript

var theValue = ko.observable("theValue");
html

<input data-bind="value: theValue, valueUpdate: 'keyup'"></input>
Output

The "hasFocus" binding

javascript

var isSelected = ko.observable(false);
html

<div class="ui form">
    <div class="inline field">
        <input type="text" data-bind="hasFocus: isSelected"></input>
        <span data-bind="visible: isSelected">输入框获得了焦点</span>
    </div>
</div>
<button class="ui orange mini button" data-bind="click: function () { isSelected(true); }">获得焦点</button>
Output
输入框获得了焦点

checkbox元素
实例中的 toggleValue 依赖 ko.observableArray.patch.js

javascript

var checkboxChecked = [];
html

<!-- 原生 方法 -->
<div data-bind="inline field">
    <input type="checkbox" value="zhangsan" data-bind="checked: checkboxChecked">张三</input>
    <input type="checkbox" value="lisi" data-bind="checked: checkboxChecked">李四</input>
    <input type="checkbox" value="wangwu" data-bind="checked: checkboxChecked">王五</input>
</div>

<!-- Semantic 方式 -->
<div class="inline field">
    <div class="ui checkbox" data-bind="click: function () { checkboxChecked.toggleValue('zhangsan'); }">
        <input type="checkbox" class="hidden" value="zhangsan" data-bind="checked: checkboxChecked">
        <label>张三</label>
    </div>
    <div class="ui checkbox" data-bind="click: function () { checkboxChecked.toggleValue('lisi'); }">
        <input type="checkbox" class="hidden" value="lisi" data-bind="checked: checkboxChecked">
        <label>李四</label>
    </div>
    <div class="ui checkbox" data-bind="click: function () { checkboxChecked.toggleValue('wangwu'); }">
        <input type="checkbox" class="hidden" value="wangwu" data-bind="checked: checkboxChecked">
        <label>王五</label>
    </div>
</div>
Output
张三 李四 王五


The "options" binding

Select 元素对应参数:
  • "option" - 数组对象
  • "optionsValue" - 设置 value 属性值。如果是数组，设置成索引值；如果是对象，设置成对应属性值。
  • "optionsText" - 设置文本属性值。如果是数组，设置成索引值；如果是对象，设置成对应属性值。
  • "optionsCaption" - 默认展示提示文本。建议不要使用，在option队列里实现具体提示项。
javascript

var selectArray = [{
    key: "aa", show: "啊啊"
}, {
    key: "bb", show: "哦哦"
}, {
    key: "cc", show: "哈哈"
}];
var selectValue = ko.observable('');
html

<!-- 建议不要使用 optionsCaption 默认值未知
<select data-bind="options: subjects, optionsValue: '0', optionsText: '1', value: subject_id, optionsCaption: '这是提示文字'"></select>
-->
<select class="ui dropdown" data-bind="options: selectArray, optionsValue: 'key', optionsText: 'show', value: selectValue"></select>
<label data-bind="text: selectValue"></label>
Output
